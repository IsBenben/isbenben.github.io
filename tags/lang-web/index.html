猜词游戏

这是一个基于Web的猜词游戏，支持键盘输入字母，实时显示统计信息。
游戏内置约5000个单词库，可将当前单词编码为分享链接，实现单词的便捷传播与挑战。
页面采用响应式布局，适配PC与移动端，提供流畅的猜词体验。

单词接龙

单词接龙是一个经典的文字游戏，只要新单词的首字母和前一个单词的尾字母相同，就可以接上。
这个网页版游戏支持不会时AI辅助和释义查询。

快速添加到收藏夹

这是一个 Chrome 浏览器插件，需要开启开发者模式导入该插件，在 B 站视频中就可以快速添加到收藏夹。
快速添加到收藏夹：根据B站视频的标签自动创建并匹配收藏夹。
原理：通过 JS 脚本操作 DOM 元素，模拟点击按钮和输入，实现快速添加到收藏夹。
