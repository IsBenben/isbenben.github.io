## 介绍

用Python为Scratch做的编程语言。包括词法分析、语法分析、编译器。

编译成品为Scratch可直接识别的project.json。

已经支持基本的语法。包括注释、函数、变量、判断、运算、列表、循环。

## 使用
```bash
git clone https://github.com/IsBenben/Scratch-Language.git
cd Scratch-Language
python src/cmdnew.py -h
```
